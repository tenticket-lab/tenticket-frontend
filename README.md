# TenTicket Frontend

Frontend application for the TenTicket platform.

---

## Tech Stack

- React
- Vite
- React Router
- Tailwind CSS
- Recharts
- Axios

---

## Features

- User Authentication
- Admin Dashboard
- Reports & Analytics
- Ticket Management
- QR Verification
- Responsive Design
- Role-Based Access Control
